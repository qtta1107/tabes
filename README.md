# 🎮 Totally Accurate 3D - Multi-Map Engine

Chào mừng đến với tựa game chiến thuật kết hợp góc nhìn thứ nhất (FPS)!

👉 **[BẤM VÀO ĐÂY ĐỂ CHƠI GAME NGAY](Dán-link-trang-web-của-bạn-vào-đây)** 👈

## 📖 Hướng dẫn cách chơi
- Dùng chuột để xoay góc nhìn.
- Bấm **WASD** để di chuyển.
- Bấm **Click Trái** để tấn công.
- Bấm **ESC** để thoát chế độ nhập vai.

## 🛠 Tính năng nổi bật
- 9 Bản đồ chiến thuật với cơ chế môi trường và vật lý khác nhau.
- Tính năng điều khiển trực tiếp một đơn vị lính (Possession System).
